Preguntas-4 PRISMA Y TYPEORM

# 1. ¿Qué es Prisma y TypeORM? 

==a) Prisma es un ORM (Object-Relational Mapping) para Node.js, mientras que TypeORM es una biblioteca para gestionar bases de datos relacionales. ==

b) Prisma y TypeORM son frameworks para el desarrollo de aplicaciones web en Python. 

c) Prisma y TypeORM son bibliotecas para el manejo de bases de datos NoSQL en Node.js.

~~~
Respuesta correcta: a) Prisma es un ORM para Node.js, mientras que TypeORM es una biblioteca para gestionar bases de datos relacionales.

Explicación: Prisma y TypeORM son herramientas que permiten a los desarrolladores interactuar con bases de datos relacionales utilizando lenguajes de programación como TypeScript o JavaScript en el caso de Node.js. Ambas son soluciones populares para el mapeo objeto-relacional en el ecosistema de JavaScript.
~~~


# 2. ¿Cuál de las siguientes afirmaciones es cierta sobre Prisma y TypeORM? 

a) Prisma y TypeORM son exclusivamente para bases de datos SQL. 

b) Prisma solo es compatible con bases de datos MySQL, mientras que TypeORM es compatible con varios tipos de bases de datos SQL. 

==c) Prisma y TypeORM son compatibles con varios tipos de bases de datos SQL.==

~~~
Respuesta correcta: c) Prisma y TypeORM son compatibles con varios tipos de bases de datos SQL.

Explicación: Tanto Prisma como TypeORM son compatibles con varios tipos de bases de datos SQL, como MySQL, PostgreSQL, SQLite, etc. Esto brinda a los desarrolladores flexibilidad para elegir la base de datos que mejor se adapte a sus necesidades.
~~~


# 3. ¿Cuál de las siguientes afirmaciones describe mejor la sintaxis para definir modelos en Prisma y TypeORM?

==a) Prisma utiliza una sintaxis basada en archivos de esquema (schema) definidos en un lenguaje específico de dominio (DSL), mientras que TypeORM utiliza decoradores de clases en TypeScript para definir modelos.==

b) Prisma y TypeORM ambos utilizan una sintaxis similar basada en decoradores de clases en TypeScript.

c) Prisma y TypeORM requieren la definición manual de SQL para cada modelo en la base de datos.

~~~
Respuesta correcta: a) Prisma utiliza una sintaxis basada en archivos de esquema (schema) definidos en un lenguaje específico de dominio (DSL), mientras que TypeORM utiliza decoradores de clases en TypeScript para definir modelos.

Explicación: Prisma utiliza un lenguaje específico de dominio (DSL) en archivos de esquema para definir los modelos y la estructura de la base de datos. Por otro lado, TypeORM emplea decoradores de clases en TypeScript para definir los modelos, lo que facilita la integración con el código de la aplicación.
~~~
